# Obstacle Assault

Projeto de estudo desenvolvido em **Unreal Engine 5 + C++** como parte de um **curso de Unreal Engine + C++**.

O objetivo do projeto é praticar programacao de gameplay, organizacao de codigo em modulos e criacao de diferentes estilos de fase dentro do mesmo jogo.

## Sobre o projeto

`Obstacle Assault` e um game/prototipo com foco em desafios, movimentacao e interacao com obstaculos, explorando mecanicas implementadas em **C++** e integradas com recursos da **Unreal Engine**.

Pelo conteudo atual do projeto, existem diferentes variantes e sistemas de gameplay, incluindo:

- `Platforming`
- `SideScrolling`
- `Combat`
- plataformas moveis
- interacoes com checkpoints, inimigos e objetos do cenario
- interfaces e elementos de UI com `UMG`

## Tecnologias utilizadas

- `Unreal Engine 5.6`
- `C++`
- `Enhanced Input`
- `UMG`
- `AIModule`
- `StateTree`
- `GameplayStateTree`

## Como abrir o projeto

1. Clone este repositorio.
2. Abra o arquivo `ObstacleAssault.uproject` pela Unreal Engine 5.6.
3. Caso necessario, gere os arquivos de projeto do Visual Studio.
4. Compile o projeto pelo Visual Studio ou diretamente pela Unreal.

## Requisitos

- `Unreal Engine 5.6`
- `Visual Studio 2022`
- workload de desenvolvimento em `Desktop development with C++`

## Objetivos de estudo

Este projeto foi usado para praticar:

- movimentacao e controle de personagem
- logica de gameplay em C++
- organizacao de codigo por variantes de jogo
- IA basica com `AIModule` e `StateTree`
- integracao entre Blueprint e C++
- criacao de niveis e interacoes no editor da Unreal

## Versionamento

O repositorio foi preparado para versionar apenas os arquivos realmente necessarios do projeto, evitando incluir pastas geradas automaticamente pela Unreal Engine, como build intermediario, cache e arquivos locais do editor.
